# Premium Section Composition Reference

## Goal
Every major section should feel cinematic and connected.

## Recommended Structure
- Full-width background image or glow layer
- Large bold headline
- Short athletic copy
- Layered product image
- Floating accent elements
- Subtle motion
- Fade transition into next section

## Layout Style
Avoid:
- Centered generic cards
- White empty space
- Simple rectangles
- Hard section cuts

Use:
- Overlapping sections
- Offset content
- Asymmetrical layouts
- Large imagery
- Glow overlays
- Layered depth
- Cinematic spacing

## Visual Feel
Reference feeling:
- Premium sports-tech
- Athlete-first
- Dark and immersive
- Modern performance brand
- Nike-style motion direction

## Image Integration
Images should:
- Bleed outside containers
- Cast glow or shadows
- Float slightly
- Layer over backgrounds
- Connect visually across sections

## Motion Integration
Each section should contain:
- Fade-in movement
- Slight parallax
- Hover depth
- Background motion
- Lighting continuity

## Typography Rules
Use:
- Large strong headlines
- Short copy blocks
- Wide spacing
- High contrast

Avoid overly long text inside visual sections.
