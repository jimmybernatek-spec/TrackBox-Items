# TrackBox Products and Boxes

## Main Box Structure
TrackBox uses a tiered premium athlete box system.

Current concepts:
- Essentials
- Performance + Recovery
- Premium

Each box is built around athlete preparation.

## Core Product Categories
- Fuel
- Hydration
- Recovery
- Competition preparation
- Consumables
- Apparel
- Training accessories

## Session Packs
Session Packs are sport-specific performance systems.

Structure:
- PRE
- DURING
- POST
- RECOVERY
- COMPETE

Examples:
- Speed Day
- Meet Day
- Pole Vault Session
- Recovery Day
- Sprint Session

## GoBags
GoBags contain snack and consumable systems.

Examples of naming directions:
- LOCK IN
- STAY FAST
- REBUILD
- COMPETE

## Packaging Direction
Packaging should feel premium.

Preferred setup:
- Matte black boxes
- Black tissue paper
- Premium inserts
- Sticker seals
- Organized product layouts

## Website Product Direction
Products should be customizable through guided systems.

Example flow:
- Select box tier
- Select chalk type
- Select PRE fuel option
- Select DURING hydration option
- Select POST recovery option

The goal is personalization while maintaining TrackBox performance structure.

## Long-Term Vision
TrackBox evolves into a full athlete preparation ecosystem integrating:
- Products
- Coaching
- Recovery systems
- Team sales
- Camps
- Digital education
- Session tracking
- Athlete readiness systems
