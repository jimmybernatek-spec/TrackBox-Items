# TrackBox Cinematic Asset Library

## Goal
Create a centralized cinematic asset system so Codex can build cohesive premium visuals instead of generic layouts.

## Main Asset Categories
Create and organize assets into:

public/assets/trackbox/backgrounds
public/assets/trackbox/products
public/assets/trackbox/3d
public/assets/trackbox/lighting
public/assets/trackbox/particles
public/assets/trackbox/textures
public/assets/trackbox/athletes
public/assets/trackbox/packaging
public/assets/trackbox/mockups
public/assets/trackbox/ui-reference

## Background Assets
Use:
- Dark track textures
- Stadium lighting
- Black gradients
- Fog or haze overlays
- Matte black surfaces
- Dark athletic environments
- Training atmosphere scenes

Preferred mood:
- Cinematic
- Premium
- Minimal
- High contrast

## Product Assets
Products should be:
- Transparent PNG cutouts when possible
- High resolution
- Professionally lit
- Centered and isolated
- Consistent lighting direction

Products may include:
- Chalk
- Athletic tape
- Hydration packets
- Snack GoBags
- Shirts
- Spike replacements
- Wrist guards
- Recovery products

## 3D Assets
Use these for homepage and cinematic scenes:
- TrackBox closed box model
- TrackBox open box model
- Floating product models
- Orbit scene objects
- Lighting rigs
- Camera paths

## Lighting Assets
Preferred lighting:
- Neon blue glow
- White rim lighting
- Dark shadows
- Stadium light streaks
- Matte reflections
- Product edge highlights

Avoid flat lighting.

## Particle Assets
Use subtle particles such as:
- Chalk dust
- Floating debris
- Atmospheric haze
- Light particles
- Motion blur streaks

Do not overuse particles.

## Texture Assets
Use:
- Carbon textures
- Rubber track textures
- Matte black surfaces
- Grain overlays
- Soft noise textures
- Metallic detail textures

## Athlete Assets
Preferred athlete images:
- Dynamic movement
- Sprinting
- Pole vaulting
- Recovery moments
- Training focus
- Dark or controlled lighting
- Minimal clutter

## Mockup Assets
Use mockups for:
- Apparel
- Packaging
- Sponsor placement
- Session Packs
- Event activations
- Product spreads

## UI Reference Assets
Store inspiration screenshots and layout references here.

Examples:
- Athletic brands
- Premium ecommerce layouts
- Motion examples
- Section structure references
- Typography references

## Codex Instructions
Codex should prioritize these uploaded assets instead of generating placeholder visuals whenever possible.

Build layouts around cinematic imagery and layered visual depth.

Do not treat images as simple rectangular content blocks.

Use assets to create immersion and motion.
