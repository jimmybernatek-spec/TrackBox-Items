# TrackBox UI System

## Goal
Make the website feel premium, cohesive, cinematic, and athletic instead of blocky or generic.

Every page should feel like one connected performance brand system.

## Overall Style
TrackBox UI should feel:
- Premium
- Dark
- Athletic
- Cinematic
- Scientific
- Minimal
- High contrast
- Product-focused
- Motion-driven

## Color System
Primary:
- Matte black backgrounds
- Deep charcoal panels
- White typography
- Cool gray secondary text

Accents:
- Neon blue for performance, technology, hydration, and active states
- Neon pink for premium highlights, energetic CTAs, and visual contrast
- Red only for intense product breakdown pages or warning-style emphasis

Avoid:
- Random bright colors
- Flat gray card stacks
- White-box generic ecommerce layouts
- Unstyled default buttons

## Typography
Use bold athletic hierarchy.

Headlines:
- Large
- Condensed or strong sans-serif feel
- High contrast
- Short punchy statements

Body copy:
- Clean
- Minimal
- Easy to scan
- Athlete-focused

Use short copy blocks. Avoid long paragraphs inside visual sections.

## Layout Rules
Avoid plain stacked rectangles.

Use:
- Layered sections
- Overlapping images
- Product cutouts
- Background glows
- Gradient fades
- Angled or masked image areas
- Full-width cinematic sections
- Large negative space
- Wide desktop layouts

Each section should connect visually to the next section.

## Card System
Cards should feel premium, not default.

Card rules:
- Dark translucent backgrounds
- Thin gray or blue borders
- Subtle inner glow
- Rounded corners
- Soft depth shadows
- Hover lift
- Icon or image integration
- No generic white cards

## Image Treatment
Images should be integrated using:
- Vignette overlays
- Background blur
- Gradient masks
- Glow behind products
- Floating product shadows
- Parallax movement
- Cutout product layering
- Dark track or stadium textures

Avoid simply dropping rectangular images into boxes.

## Section Connector Rules
Between sections, use:
- Fade overlays
- Gradient transitions
- Shared background texture
- Product/image overlap
- Floating particles
- Continuation of light source
- Scroll-based motion

The website should not feel like disconnected blocks.

## Button System
Primary CTA:
- Dark or neon-accented
- Strong border or glow
- Short text
- Hover animation

Secondary CTA:
- Transparent
- White or gray border
- Subtle glow on hover

Button text examples:
- Build Your TrackBox
- Choose Your Box
- View Session Packs
- Partner With TrackBox

## Homepage Flow
Preferred homepage structure:
1. 3D scroll intro
2. Cinematic hero continuation
3. Choose Your TrackBox
4. What is inside
5. Session Packs
6. Sponsors and partners
7. Pole Vault Miami integration
8. Team sales
9. Final CTA

## Codex Instructions
When building UI, avoid generic templates.

Always use:
- Real uploaded assets when available
- Dark cinematic image treatments
- Premium spacing
- Consistent section rhythm
- Responsive mobile simplification
- Reusable components

If images are uploaded, build around them. Do not ignore them.
