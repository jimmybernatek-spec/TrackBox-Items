# TrackBox Sponsors Section

## Goal
Build a full editable sponsors and partners section for the TrackBox website.

This section should make TrackBox look premium, organized, and partnership-ready.

Important: Do not claim that any company is an official sponsor unless that company is already confirmed by TrackBox.

## Website Locations
Use this system in:
- Homepage sponsor strip
- Dedicated Sponsors page
- Product pages
- Event pages
- Footer logo strip

## Section Headlines
Use options like:
- Built With Performance Partners
- Partners In Preparation
- The Brands Behind Better Preparation
- Fueling The Next Generation Of Track Athletes

## Sponsor Categories
Organize sponsors by:
- Hydration
- Recovery
- Nutrition
- Fuel
- Gear
- Apparel
- Pole Vault Equipment
- Track and Field Equipment
- Technology
- Athlete Services
- Local Business Partners
- Event Partners

## Sponsor Tiers
Create editable tiers:

### Title Partner
Top placement. Large feature card, homepage display, and premium page placement.

### Performance Partner
Strong placement for brands connected to athlete preparation, recovery, fuel, hydration, gear, or technology.

### Box Partner
For brands or products included inside TrackBox boxes or Session Packs.

### Event Partner
For camps, clinics, meets, activations, or local events.

### Community Partner
For local partners supporting athlete development.

## Sponsor Card Fields
Each card should support:
- Sponsor name
- Category
- Tier
- Short description
- Why this sponsor fits TrackBox
- Logo path
- Featured image path
- Product image path
- Website URL
- Instagram URL
- Discount code
- Partnership status
- Featured true or false

## Image Folder Structure
Create and use these folders:

public/assets/trackbox/sponsors/logos
public/assets/trackbox/sponsors/products
public/assets/trackbox/sponsors/banners
public/assets/trackbox/sponsors/events

## Image Rules
- Use transparent PNG or SVG logos when possible.
- Do not stretch logos.
- Keep spacing consistent.
- Use dark premium card backgrounds.
- Use placeholders when real assets are missing.
- Clearly mark placeholder assets.

## Sponsor Criteria
Only feature brands that fit TrackBox through at least one of these:
- Helps athletes prepare better
- Supports hydration, fuel, recovery, gear, or performance
- Supports track and field athletes
- Supports camps, teams, clinics, or events
- Fits TrackBox Session Packs or product systems
- Adds real value to athletes

## Homepage Section Layout
1. Small label: Performance Partners
2. Headline: Built With Athlete-First Brands
3. Short paragraph explaining the partner standard
4. Horizontal logo strip
5. Featured partner cards
6. Button: Become a TrackBox Partner

## Dedicated Sponsors Page Layout
1. Hero section
2. Sponsor mission statement
3. Featured partner area
4. Category filter system
5. Sponsor card grid
6. Event partner section
7. Partnership criteria section
8. Contact or inquiry call to action

## Design Direction
Match the TrackBox brand:
- Matte black
- White typography
- Gray dividers
- Neon blue accents
- Neon pink accents
- Premium logo cards
- Subtle hover animations
- Clean grid layout
- High contrast
- Minimal clutter

## Codex Build Instructions
Build this as a reusable section and, if possible, a full sponsors page.

Use a data-driven structure so sponsors can be edited later without rebuilding the layout.

If using code, create a sponsor data file with sponsor objects.

If using a visual builder or CMS, map the same fields into editable CMS items.

Do not invent official sponsorships or partnership claims.
