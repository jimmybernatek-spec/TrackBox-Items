# Active Theory and Resn Reference Direction

## Goal
Use Active Theory and Resn as the primary interaction and art-direction references for the TrackBox website.

TrackBox should feel like an immersive interactive experience, not a normal ecommerce site.

## Reference Studios
Primary references:
- Active Theory
- Resn

These references should guide:
- interaction design
- scroll-based storytelling
- 3D movement
- atmospheric motion
- cinematic transitions
- immersive page flow
- cursor and hover behavior
- product reveal systems
- scene-to-scene transitions

## Desired Feeling
The TrackBox site should feel:
- Experimental but clean
- High-tech
- Premium
- Immersive
- Cinematic
- Athletic
- Dark
- Motion-heavy
- Spatial
- Interactive

It should feel closer to an interactive digital experience than a normal product landing page.

## Interaction Direction
Use:
- Scroll-controlled scenes
- Camera movement
- Parallax depth
- Cursor-reactive objects
- Smooth transitions
- Floating product systems
- 3D product reveals
- Layered atmospheres
- Scene-based homepage sections
- Interactive cards and panels
- Animated sponsor/product systems

Avoid:
- Plain vertical stacked sections
- Basic cards
- Simple fade-ins only
- Static ecommerce layouts
- Hard cuts
- Generic templates

## Homepage Experience
The homepage should be built as an immersive scroll journey.

Sequence:
1. Dark environment loads with subtle motion.
2. Camera moves toward the TrackBox.
3. TrackBox appears as the central 3D object.
4. Box rotates subtly with lighting.
5. Scroll opens the box.
6. Products fly out and orbit.
7. Product callouts appear in space.
8. User continues into interactive sections.
9. Normal information appears only after the cinematic intro.

## Visual Direction
Use:
- Dark backgrounds
- Matte black surfaces
- Neon blue accents
- Neon pink accents
- Stadium haze
- Track textures
- Chalk particle systems
- Soft glows
- Light streaks
- Depth blur
- Floating panels

## UI Direction
The UI should feel like it exists inside the scene.

Use:
- Floating panels
- Glassy dark cards
- Hover-responsive elements
- Depth layers
- Motion-linked text
- Section transitions that feel continuous

## Motion Direction
Motion should be constant but controlled.

Use:
- GSAP ScrollTrigger
- Three.js
- Framer Motion
- Spline if needed

Motion should feel premium, not playful.

## Codex Instructions
When rebuilding the website, use this document as a major visual and interaction source of truth.

Do not copy any proprietary code or exact creative work.

Use the references as inspiration for the level of polish, interactivity, motion, and immersive feel.

The final TrackBox site should be original, but it should match the immersive, high-end, interactive quality level of Active Theory and Resn.
